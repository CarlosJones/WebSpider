内容简介 ······

本书讲述的是计算机工作原理。作者用丰富的想象和清晰的笔墨将看似繁杂的理论阐述得通俗易懂，你丝毫不会感到枯燥和生硬。更重要的是，你会因此而获得对计算机工作原理较深刻的理解。这种理解不是抽象层面上的，而是具有一定深度的。

作者简介 ······

Charles Petzold从1984年开始编写个人计算机程序，从1985年开始编写 Microsoft Windows程序。他在《Microsoft Systems Joumal》1986年 12月号上发表了第一篇关于Windows程序设计的杂志论文。从1986年到1995年，他为《PC Magazine》撰写“Environments”专栏，给读者介绍了Windows和OS／2程序设计的许多方面。

1994年5月，Petzold作为仅有的七个人之一（并且是唯一的作家）被《Window Magazine》和Microsoft公司授予Windows Pioneer奖，以表彰他对 Microsoft Windows的成功做出的贡献。

    作为一个计算机爱好者，花了太多的时间在操作系统的维护、软件的使用上。在安装装系统、使用软件的时候，总是有一种疑问——计算机怎样将电能转化为计算的能力，电信号是怎样在计算机中运行的。而在学习C语言编程的时候，花了很大精力都没有办法理解指针、地址的意思。
    进入大学以后，虽然学习的同计算机和电子没有什么关系，可是还是想彻底的理解计算机的运行机制，看了很多书，从《计算机组成原理》到《深入理解计算机体系》。虽然都是非常经典的书，可惜还是因为相关知识了解太少而徒劳无功。偶尔看了这本书《编码：隐匿在计算机软硬件背后的语言》，感觉豁然开朗。很多关于计算机运行、计算机现状的疑问在本书都得到了解答。看完这本书之后，再看《深入理解计算机体系》、《数字电路》，理解起来非常容易。

    第1章 至亲密友
    作者用两个互为邻居的孩子之间通讯做例子说明编码的概念：指一种用来在机器和人之间传递信息的方式。编码就是交流。各种编码方式都有其优点和缺点，比如声音不能储存在纸上，在黑暗情况下书写无法交流等。如果一种编码可以用在其它编码无法取代的地方，那么它就是一种有用的编码。两个不同的事物，只要经过适当的组合，就可以表示所有类型的信息。
    第2章 编码与组合
    含有两个元素的编码，码字的数目=2^编码的位数

    第3章 布莱叶盲文与二进制码
    用六个凹或凸的点来表示一个字母/数字/符号，共有2^6=64个不同的组合。
    其中数字标识符和取消“数字标识状态”的字母标识符，改变了后面编码的意义，这样的编码称为“优先码(Precedence codes)”或“换档码(Shift codes)”，它们改变着作用域内编码的含义，直到作用域结束。
    大写字母标识符表示紧随它的字母应该被译为大写。这被称为“逃逸码（Escape codes）”。

    第10章 逻辑与开关
    在布尔代数中，“+”（OR）表示两个集合的并集，两个集合的并集就是指第一个集合中的所有元素与第二个集合中所有元素的集合。
    “X”（AND）表示两个集合的交集，指即在第一个集合中又在第二个集合中的所有元素的集合。
    结合律： A+(BXC) = (A+B)X(A+C) AX(B+C) = (AXB)+(AXC)

    第11章 门
    两个继电器串联称为与门，其真值与AND一样。
    两个继电器并联称为或门，其真值与OR一样。

    第12章 二进制加法器
    就是找到一个电路，使其真值表与加法的真值表一致。
    加法结果的真值表
    + 0 1
    0 0 1
    1 1 0
    进位 0 1
    0 0 0
    1 0 1
    异或门
    XOR 0 1
    0 0 1
    1 1 0
    二进制加法器是由一个XOR和一个AND并联后形成的。这叫半加器，因为没有考虑多位数加法的进位。
    将两个半加器串联，第一个的结果和进位输入作为第二个半加器的输入，第一个半加器的进位输出和第二个半加器的进位输出进行或运算，输出为全加器的进位输出，第二个半加器的结果作为全加器的和输出。多个全加器串联就可以进行多位二进制数的加法运算。
    第13章 如何实现减法
    关键问题：避免借位
    例: 253-176 = 253 - 176 + 1000 - 1000 = 253 - 176 + 999 + 1 - 1000 = 253 + (999-176) + 1 - 1000 这样就没有借位问题了。
    二进制减法步骤：1.用11111111减去减数。2.将减数对1的补数与被减数相加。3.将上式所得结果加1.4.减去100000000（第一步的数再加1)
    补数（补码）的作用，将在一定范围内的负数转换为正数。
    对十进制，在-500到500的范围内，-500的补数为500 (算法:999-500+1)，对二进制的补数，是将各位取反再加1.例如十进制125的二进制为01111101，其对2的补数为，先将每位取反得10000010，再加1得10000011。这个系统提供了一个不用负号就能表示正、负数的方法，并可自由的将正数和负数用加法法则相加。
    这样同同一个电路就可以计算加法和减法。
    第14章 反馈与触发器
    反馈就是由一个反向器组成的闭合回路，当闭合时，输入均被取反输出然后作为新的输入返回，如此往复。这样的电路叫振荡器。
    两个门原件，一个的输出是另一个的输入，另一个的输出又是其输入，这就是反馈。
    由两个或非门组成的反馈电路称为触发器(Flip-Flop)。触发器电路可以保持信息，可以“记住”某些信息。这是R-S(Reset-Set)触发器。
    从其的输出状态可以判断最后闭合的开关是哪个。对R-S触发器，禁止两个开关同时闭合。两个开关都打开时，输出状态不变。
    在RS触发器基础上，在其前面增加两个与门，并增加一个保持位，同时与复位和置位相连。当保持位为1时，输入有效。否则输入无效。
    进一步简化，可以将R和S位合并为一个数据位，其中一端与反向器相连。这样的触发器叫电平触发的D型触发器（D代表数据）。保持位会持续的反复变化，所以也叫时钟。这个电路即所谓的电平触发的D型锁存器，它表示电路锁存住一位数据并保持它，以便将来使用。这个电路也可以被称为1位存储器。多个1位锁存器连在一起构成多位锁存器。有了存储器，可以把存储器的输出作为加法器的输入，从而进行多次加法。用一个2-1选择器选择从用户输入还是从存储器输入。
    改进：增加一个或门和一个清零的输入信号，当其为1时，锁存器输出为0.
    边沿触发的D型触发器，由两级R-S触发器连接而成。第二级RS触发器的输出是第一级的输入，只有当时钟信号由0变为1时，数据端输入才被保存下来。
    把振荡器的输出与边沿触发的D型触发器的时钟端输入连接，同时把Q'端输出连接到本身的输入端，其输出又是自己的输入。当多个这样的触发器串联时，每个触发器的频率依次减半，构成了一个行波计数器。

    第17章 自动操作

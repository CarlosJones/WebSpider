    多核的内存结构
    UMA（Uniform Memory Access）：最简单的内存共享方式就是将内存作为与执行核独立的单元构建在核之外，所有的核通过同一总线对内存进行访问。对内存的竞争将变得白热化，从而造成系统效率急剧下降。
    NUMA（Non-Uniform Memory Access）：使用多个独立的共享内存。
    COMA（Cache Only Memory Access）：每个执行核里面配置缓存，其执行需要的数据均缓存在该缓存里面。
    NORMA（Non - Remote Memory Access）：如果内存单元为每个执行核所私有，且每个执行核只能访问自己的私有内存，对其他内存单元的访问通过消息传递进行，则就是所谓的非远程内存访问模式。

    多核的内存结构
    UMA（Uniform Memory Access）：最简单的内存共享方式就是将内存作为与执行核独立的单元构建在核之外，所有的核通过同一总线对内存进行访问。对内存的竞争将变得白热化，从而造成系统效率急剧下降。
    NUMA（Non-Uniform Memory Access）：使用多个独立的共享内存。
    COMA（Cache Only Memory Access）：每个执行核里面配置缓存，其执行需要的数据均缓存在该缓存里面。
    NORMA（Non - Remote Memory Access）：如果内存单元为每个执行核所私有，且每个执行核只能访问自己的私有内存，对其他内存单元的访问通过消息传递进行，则就是所谓的非远程内存访问模式。
